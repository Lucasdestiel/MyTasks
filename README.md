
Uma aplicação moderna de gerenciamento de tarefas desenvolvida com HTML, CSS e JavaScript, criada para ajudar usuários a organizarem suas atividades diárias de forma simples, rápida e intuitiva.

✨ Funcionalidades
✅ Adicionar novas tarefas
✏️ Editar tarefas existentes
🗑️ Excluir tarefas
✔️ Marcar tarefas como concluídas
🔍 Filtrar por status (Todas, Pendentes e Concluídas)
📊 Estatísticas em tempo real
📈 Barra de progresso das tarefas
💾 Armazenamento local com Local Storage
📱 Layout responsivo para dispositivos móveis
🚀 Tecnologias Utilizadas
HTML5
CSS3
JavaScript (ES6+)
Local Storage API
🎯 Objetivo do Projeto

O objetivo deste projeto foi praticar conceitos fundamentais do desenvolvimento Front-End, incluindo:

Manipulação do DOM
Gerenciamento de estado da aplicação
Persistência de dados no navegador
Criação de interfaces responsivas
Boas práticas de organização de código
Experiência do usuário (UX)
